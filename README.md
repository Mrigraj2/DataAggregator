# 📦 DataAggregator

**DataAggregator** is a lightweight Java library that connects to a relational database using JDBC and exports SQL query results to an Excel `.xlsx` file using Apache POI.

---

## ✅ Features

- Connect to My-Sql JDBC database
- Run SQL queries
- Export query results to Excel with column headers
- Simple and extensible API

---

## 🛠️ Installation

Add this library to your project via Maven:

```xml
<dependency>
    <groupId>com.dataaggregator</groupId>
    <artifactId>data-aggregator</artifactId>
    <version>1.0.0</version>
</dependency>
